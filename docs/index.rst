
Teleport VR
===========

These pages document Teleport VR: the Spatial Internet Protocol, its reference implementation,
tools and SDK's. The documentation is given in the likely order that
most users will approach Teleport: firstly, an :doc:`introduction </introduction>` to the ideas and 
concepts behind the system. Secondly, a guide to the default Teleport :doc:`Client</client/index>` app.
Thirdly, a guide on using the :doc:`Unity SDK </unity/index>` for setting
up a virtual reality server. Fourthly, the documentation of the :doc:`Reference Implementation</reference/index>` ,
which can be used to build Teleport clients or servers. And finally,
the official :doc:`specification </protocol>` of the Teleport Protocol.
